# Personal Academic Homepage Template

This is a lightweight static template for a personal academic homepage.

## Customize

- Edit `index.html` and replace placeholder text such as `Your Name`, `Your Position`, `Professor Name`, and sample publications.
- Replace `profile.svg` with your own profile photo, or update the image path in `index.html`.
- Update contact links, research interests, education, publications, experience, teaching, and awards.
- The default affiliation is Fudan institute of Systems for Advanced Computing (FiSAC), Fudan University.
